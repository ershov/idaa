Layout Settings



  • About

    Layout and usability improvement Chrome extension for various sites.

    Pick a site on the left to configure the layout.

    "Dark Mode" and "Reduce Animation" checkboxes at the top will change the setting for all sites.

    [Save] button syncs the settings to the Profile that's accessible across devices. Closing the dialog also saves settings.

    [Undo] undoes the changes to the point when Settings were opened.

    [Defaults] loads defaults (everything on, smart Dark Mode).

    The extension tries to be smart about using Dark Mode and gets the default preference from System.

    Items marked with «☼» use script and may require the page refresh in order to take effect. That's automatic upon dialog close or [Save].

    Chat and Contacts are widely embedded into other sites (gmail, buganizer, etc). In order to control their embedded behavior, go to their tabs.

    Use keyboard shortcuts to quickly switch the dark mode or call the config window.
